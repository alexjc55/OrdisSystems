КРИТИЧЕСКИЕ КОМАНДЫ ДЛЯ VPS СЕРВЕРА (ordis.co.il)
=======================================================

ВЫПОЛНИТЬ В ТОЧНОМ ПОРЯДКЕ:

1. Подключиться к VPS:
   ssh ordis.co.il

2. Перейти в директорию проекта:
   cd /var/www/edahouse.ordis.co.il

3. ВАРИАНТ A - Быстрое исправление (1 минута):
   pm2 stop edahouse
   psql -h localhost -U edahouse_usr -d edahouse -c "ALTER TABLE store_settings ADD COLUMN IF NOT EXISTS slider_autoplay BOOLEAN DEFAULT true;"
   psql -h localhost -U edahouse_usr -d edahouse -c "UPDATE store_settings SET slider_autoplay = true WHERE id = 1;"
   pm2 start edahouse

4. ВАРИАНТ B - Если база данных недоступна:
   pm2 stop edahouse
   git reset --hard HEAD~3
   pm2 start edahouse

5. Проверка результата:
   pm2 logs edahouse --lines 10
   curl http://localhost:3000/api/settings

ОШИБКА: column "slider_autoplay" does not exist
ПРИЧИНА: Отсутствует колонка в таблице store_settings
РЕШЕНИЕ: Добавить колонку через ALTER TABLE

После исправления ошибки в логах исчезнут!
