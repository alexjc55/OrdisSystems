# Быстрая установка нового сайта

**Условия:** сайт создан в FastPanel, БД создана, GitHub синхронизирован, SSH подключён.

---

## 1. Клонируем проект

Найдите путь к корневой папке сайта в FastPanel (обычно это `/var/www/demo.ordis.co.il`).

```bash
cd /var/www/demo.ordis.co.il
git clone https://github.com/ВАШ_ЛОГИН/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git .
```

> Точка в конце — обязательна, она означает «клонировать в текущую папку».

---

## 2. Устанавливаем пакеты и собираем

```bash
npm install
npm run build
```

Ждём завершения. В конце должен появиться файл `dist/index.js`:

```bash
ls dist/
```

---

## 3. Создаём файл настроек

```bash
nano .env
```

Вставляем содержимое (замените значения на свои):

```env
NODE_ENV=production

# Порт — тот, что назначен сайту в FastPanel
PORT=5003

# База данных
DATABASE_URL=postgresql://ПОЛЬЗОВАТЕЛЬ_БД:ПАРОЛЬ@localhost:5432/НАЗВАНИЕ_БД
PGHOST=localhost
PGPORT=5432
PGDATABASE=НАЗВАНИЕ_БД
PGUSER=ПОЛЬЗОВАТЕЛЬ_БД
PGPASSWORD=ПАРОЛЬ

# Секретный ключ — любой длинный случайный текст (мин. 32 символа)
SESSION_SECRET=впишитездесьлюбойдлинныйслучайныйтекст1234567890abcdef

# SendGrid (если нужна отправка email, иначе оставьте пустым)
SENDGRID_API_KEY=
```

Сохраняем: `Ctrl+O` → `Enter` → `Ctrl+X`

---

## 4. Создаём структуру базы данных

```bash
npm run db:push
```

Команда автоматически создаёт все таблицы в БД. Должно завершиться без ошибок.

---

## 5. Запускаем сайт через PM2

```bash
pm2 start dist/index.js --name demo.ordis.co.il
```

Проверяем что запустился:

```bash
pm2 status
```

Статус должен быть `online` (зелёный).

Быстрая проверка что сайт отвечает:

```bash
curl http://localhost:5003/api/health
```

Ответ должен быть: `{"status":"healthy",...}`

---

## 6. Автозапуск PM2 при перезагрузке сервера

```bash
pm2 startup
```

Эта команда выведет одну строку — скопируйте и выполните её. Затем:

```bash
pm2 save
```

---

## Готово

Сайт работает. Откройте в браузере `https://demo.ordis.co.il`.

---

## Шпаргалка команд PM2

| Действие | Команда |
|---|---|
| Статус | `pm2 status` |
| Логи | `pm2 logs demo.ordis.co.il` |
| Перезапустить | `pm2 restart demo.ordis.co.il` |
| Остановить | `pm2 stop demo.ordis.co.il` |

## Обновление сайта (после изменений в GitHub)

```bash
cd /var/www/demo.ordis.co.il
git pull origin main
npm run build
pm2 restart demo.ordis.co.il
```
