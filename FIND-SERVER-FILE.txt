ПОИСК ПРАВИЛЬНОГО ПУТИ К СЕРВЕРУ НА VPS
======================================

ПРОБЛЕМА: Файл server/index.js не найден в текущей папке

КОМАНДЫ ДЛЯ ПОИСКА ФАЙЛОВ:

1. Проверить содержимое текущей папки:
ls -la

2. Найти все файлы index.js на сервере:
find /var/www -name "index.js" -type f 2>/dev/null

3. Найти папку server:
find /var/www -name "server" -type d 2>/dev/null

4. Проверить структуру проекта:
ls -la /var/www/edahouse.ordis.co.il/
ls -la /var/www/ordis_co_il_usr/data/www/edahouse.ordis.co.il/

5. Найти все файлы .js в проекте:
find . -name "*.js" -type f | head -20

6. Проверить есть ли собранные файлы (dist):
ls -la dist/
ls -la build/

ВОЗМОЖНЫЕ ВАРИАНТЫ ЗАПУСКА:
- pm2 start dist/index.js --name edahouse
- pm2 start build/index.js --name edahouse
- pm2 start index.js --name edahouse
- pm2 start app.js --name edahouse

ПОСЛЕ ОПРЕДЕЛЕНИЯ ПРАВИЛЬНОГО ПУТИ выполнить миграцию базы данных и запустить приложение.
